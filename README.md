# 👥 Real-Time People Counter

A computer vision application that detects and counts 
people in images and videos using YOLOv8 and OpenCV.

## Features
- Real-time people detection from webcam
- Image upload and analysis
- Video file processing
- Confidence score display

## Tech Stack
- Python 3.10
- YOLOv8 (Ultralytics)
- OpenCV
- Streamlit

## How to Run
pip install ultralytics opencv-python streamlit
streamlit run app.py
